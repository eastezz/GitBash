# Git Push Tutorial

## What is `git push`?

`git push` uploads your local commits to a remote repository (e.g., GitHub).

---

## Basic Push

```bash
git push
```

---

## First Push to a New Repository

```bash
git push -u origin main
```

This sets the default upstream branch.

---

## Push a Specific Branch

```bash
git push origin branch-name
```

---

## When to Use

Push when you want to:

* share your work
* back up changes online
* collaborate with others

---

## Important Tip

Always pull before pushing if others are working on the same project.

```bash
git pull
git push
```
